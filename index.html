Deploy to Azure

Follow these steps to deploy your landing zone using Azure Cloud Shell:

1
Open Azure Cloud Shell

Click the button below to launch Cloud Shell in your browser. Select Bash as the shell type.

Open Azure Cloud Shell
2
Clone the repo & switch to the hub directory

Run the following commands in Cloud Shell to clone the repository and navigate to the deployment folder:

git clone https://github.com/francisnazareth/landingzone.git
cd landingzone/hub
3
Download & upload the parameter file

Click the Download button above to save main.bicepparam to your machine. Then in Cloud Shell, click the Upload/Download toolbar button (or drag & drop) to upload it. The file will land in your home directory (~/), so move it to the correct folder:

mv ~/main.bicepparam ~/landingzone/hub/
4
Run the deployment command

Execute the following command in Cloud Shell, replacing <azure-region> with your target region (e.g., canadacentral, eastus):

az deployment sub create --location <azure-region> --template-file main.bicep --parameters main.bicepparam
